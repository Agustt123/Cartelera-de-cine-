Mi proyecto

Este proyecto fue comenzado con el fin de aprender las distintas tecnologias del backend y frontend , para ello usamos distintas tecnologias, en este proceso aprendimos los fundamentos del backend , las tecnologias de la misma , las base de datos y muchas cosas mas a lo largo del proyecto

Este Poyecto se basa en una cartelera de cine , la cual almacena las distintas peliculas que hubo y vio el cliente , para esto tiene varias secciones , la principal que es para ver las peliculas que fue agregando y la "Agregar Peliculas" la cual conecta con la Base de Datos para guardarlas ,en este se cargan distintos datos ,como el titulo ,duracion,director ,año,genero y puntuacion de la misma.
